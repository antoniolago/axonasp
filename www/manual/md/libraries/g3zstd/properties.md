# G3ZSTD Properties

## Overview
This page summarizes the properties available in the G3ZSTD library for runtime inspection.

## Properties Reference

| Property | Access | Type | Description |
|---|---|---|---|
| LastError | Read-only | String | Stores the most recent operation error message. |
| Level | Read-only | Integer | Gets the current default compression level. |

## Remarks
- Property names are case-insensitive.
- Use SetLevel to change the compression level.
