# G3MD Methods

## Overview
This page summarizes methods available in the G3MD library for Markdown processing in G3Pix AxonASP.

## Methods Reference

| Method | Returns | Description |
|---|---|---|
| Process | String | Converts Markdown source text to HTML output using the current object options. |

## Remarks
- Method names are case-insensitive.
- G3MD uses GitHub Flavored Markdown extensions.
