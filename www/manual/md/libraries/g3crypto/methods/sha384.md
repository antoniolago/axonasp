# Compute a SHA-384 Digest

## Overview

Computes the SHA-384 digest of the input and returns the result as a lowercase hexadecimal string.

## Prerequisites

Instantiate the library with `Server.CreateObject("G3CRYPTO")`.

## Syntax

```asp
result = crypto.SHA384(input)
```

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| **input** | String or Array | No | Input data as text or a VBScript byte array. When omitted, the method hashes an empty string. |

## Return Value

- **String**: 96-character lowercase hexadecimal SHA-384 digest.

## Remarks

- SHA-384 is part of the SHA-2 family and provides a larger digest than SHA-256.
- Method names are case-insensitive.

## Example

```asp
<%
Option Explicit
Dim crypto, digest
Set crypto = Server.CreateObject("G3CRYPTO")

digest = crypto.SHA384("Secure Data Transmission")
Response.Write digest

Set crypto = Nothing
%>
```

## API Reference

- **Object**: `G3CRYPTO`
- **Method**: `SHA384`
- **Arguments**: `input` (String or Array, optional)
- **Returns**: String — 96-character lowercase hexadecimal digest
