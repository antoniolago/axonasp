# Compute a SHA-512 Digest

## Overview

Computes the SHA-512 digest of the input and returns the result as a lowercase hexadecimal string.

## Prerequisites

Instantiate the library with `Server.CreateObject("G3CRYPTO")`.

## Syntax

```asp
result = crypto.SHA512(input)
```

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| **input** | String or Array | No | Input data as text or a VBScript byte array. When omitted, the method hashes an empty string. |

## Return Value

- **String**: 128-character lowercase hexadecimal SHA-512 digest.

## Remarks

- SHA-512 provides a large digest size for strong integrity requirements.
- Method names are case-insensitive.

## Example

```asp
<%
Option Explicit
Dim crypto, digest
Set crypto = Server.CreateObject("G3CRYPTO")

digest = crypto.SHA512("Maximum Security Hash")
Response.Write digest

Set crypto = Nothing
%>
```

## API Reference

- **Object**: `G3CRYPTO`
- **Method**: `SHA512`
- **Arguments**: `input` (String or Array, optional)
- **Returns**: String — 128-character lowercase hexadecimal digest
