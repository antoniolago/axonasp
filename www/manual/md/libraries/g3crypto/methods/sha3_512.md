# Compute a SHA3-512 Digest

## Overview

Computes the SHA3-512 digest of the input and returns the result as a lowercase hexadecimal string.

## Prerequisites

Instantiate the library with `Server.CreateObject("G3CRYPTO")`.

## Syntax

```asp
result = crypto.SHA3_512(input)
```

## Parameters

| Parameter | Type | Required | Description |
|---|---|---|---|
| **input** | String or Array | No | Input data as text or a VBScript byte array. When omitted, the method hashes an empty string. |

## Return Value

- **String**: 128-character lowercase hexadecimal SHA3-512 digest.

## Remarks

- SHA3-512 is a high-strength option in the SHA-3 family.
- Method names are case-insensitive.

## Example

```asp
<%
Option Explicit
Dim crypto, digest
Set crypto = Server.CreateObject("G3CRYPTO")

digest = crypto.SHA3_512("Ultra Secure Payload")
Response.Write digest

Set crypto = Nothing
%>
```

## API Reference

- **Object**: `G3CRYPTO`
- **Method**: `SHA3_512`
- **Arguments**: `input` (String or Array, optional)
- **Returns**: String — 128-character lowercase hexadecimal digest
